	Дана строка, состоящая из элементов и операторов. Операторами являются скобки 
и знаки +. Элементами являются символы (большие буквы латинского алфавита). 
Вид строки, количество элементов и операторов может быть произвольным.

Например: (X + (D+C+A) + C +(M+L + (C + H))).
Необходимо отсортировать элементы в строке по алфавиту, с учётом скобок.
В данном случае должно получиться: ((A+C+D)+C+((C+H) + L + M)+X)

Реализуйте функцию А, которая получает на вход строку данного вида и возвращает 
отсортированную строку. Считается, что синтаксис строки заведомо корректен 
(скобки парны, никакого "мусора" нет и т.п.).

Реализуйте два тестовых приложения для тестирования данной функции:

Тестовое приложение 1. Консольное.
	Должно сгенерировать N случайных строк с операторами и символами. 
	Строки должны быть как можно более разнообразными. Число N передается в 
	командной строке. Строки пишутся в файл, путь к которому так же передается в 
	командной строке.

Тестовое приложение 2. Клиент-серверное.

	Должно обрабатывать файл, сформированный тестовым приложением 1 и 
	подавать эти строки для обработки в функцию А, сохраняя результат в 
	выходной файл.

Требования к реализации:

Серверная часть на WCF с хостингом в консольном приложении. Умеет сортировать заданные строки с помощью функции А.

Клиентская часть на WinForms. Умеет взаимодействовать с сервером. Пользовательский интерфейс должен содержать следующие элементы управления:

	- текстовая строка, в которой задается путь к входному файлу
	- кнопка запуска процесса обработки
	- прогресс бар - в котором отображается ход обработки файла
	- листинг обработанных строк - так же обновляется по ходу обработки файла

Язык программирования C#, .NET Framework 4.5 и выше. Предпочтительно использовать MS Visual Studio 2015.


---------------------------------------

Решение  TestTask.BuildStrings имеет четыре проекта:
	RandomBracketStringGenerator - консольное приложение приложение - генератор строк;
	ParseLib - библиотека - обработка строк;
	ParseLibHost - приложение WCF - сервис для ParseLib;
	ParseClient - WinForms приложение - клиент для ParseLibHost
	
Порядок выполнения
	сгенерить строчки  (RandomBracketStringGenerator)
	поднять сервис (ParseLibHost)
	Выбрать в клиенте (WinForms) сгенерированный файл и запустить обработку
